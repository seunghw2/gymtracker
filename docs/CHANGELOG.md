## 2026-05-31 (UX 정리 2차)
- 홈 체중 입력: 눈금자 드래그 버그 수정(카드 터치 가로채기 제거), **저장 버튼** 추가(자동저장 폐지), 체중 팝업에서 **체지방 칸 제거**
- 체지방은 **통계 '체지방' 탭에서 별도 입력**·저장
- 운동 화면 상단: 부위 7칩 상시노출 → **"부위" 버튼 + 작은 시트 팝업** 선택, 세션 메모는 **설정 토글로 표시/숨김**
- 종목 추가: **첫 화면 상시 검색바**(이름 검색), **즐겨찾기(별표)** — 첫 화면 상단 고정(설정 저장, 두 계정 공유)
- 설정: "세션 메모 칸 표시", "체중 자동 팝업" 토글 추가
- 홈 상단 통계: [이번주 운동 / 연속 스트릭] → **[이번달 운동 / 이번주 운동]**
- 캘린더: 월 요약(이번달 운동·총 운동시간) 제거, 세션 카드 **날짜를 상단에 강조 표시**(주별 가독성↑)
- 운동 상단: 헬스장·부위를 **한 줄에 나란히**, 부위 선택을 **화면 가운데 팝업**으로(맨 아래 시트 X)
- 체중 다이얼을 **드래그 제스처 방식으로 재작성**(스크롤 방식이 모달에서 안 끌리던 문제 해결)
- 부위 목록 **사용자 편집** 가능(설정 "부위 관리" + 부위 팝업 "편집"), 설정에 저장해 두 계정 공유
- 세션 이름을 **선택한 부위로 자동**(미선택 시 날짜). 운동 헤더는 입력칸 대신 자동 이름 표시
- 부위 선택 팝업에 **건너뛰기** 버튼(미선택 시), 부위 칩 확대·순서(부위→헬스장)
