# GymTracker · 화면 캡처 (17종)

iOS 시뮬레이터(iPhone 17 Pro) 실제 앱 전체화면 캡처. 각 화면은 딥링크로 진입해 촬영.
**화면별 상세 동작·인터랙션은 [../FLOWS.md](../FLOWS.md) 참고.** 전체 흐름도는 [../SCREEN_FLOW.md](../SCREEN_FLOW.md).

| 03 | 기록 (타임라인·월별) | `(tabs)/calendar` |
| 04 | 리포트 (주/월/분기/반기) | `(tabs)/reports` |

| 06 | 설정 | `(tabs)/settings` |
| 07 | 목표 설정 | `goals` |
| 08 | 계정 | `account` |
| 09 | 운동 세션 | `workout` |
| 10 | 운동 추가(피커) | `exercise-add` |
