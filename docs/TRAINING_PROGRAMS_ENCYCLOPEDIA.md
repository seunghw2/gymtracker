# 1부. 훈련 구조 기초

명명 프로그램들은 대부분 아래 "분할 + 빈도" 틀 위에 세부 처방을 얹은 것이다. 카탈로그를 읽기 전에 이 토대를 먼저 잡는다.

## 1-1. 분할이란 무엇이고 왜 나누는가

분할(split)은 "일주일치 운동 볼륨을 며칠에 걸쳐 어떻게 쪼개 배치하느냐"의 문제다. 핵심은 다음 둘이 분리된다는 점이다.

- **주간 볼륨(weekly volume)**: 부위별로 일주일에 몇 세트를 하느냐 → 성장의 주 동력
- **빈도(frequency)**: 그 볼륨을 며칠에 나눠 배치하느냐 → 분배 수단

즉 분할 선택은 "얼마나 할 것인가"가 아니라 "어떻게 쪼갤 것인가"를 정하는 작업이다.

## 1-2. 주당 빈도와 근비대 — 빈도는 볼륨을 나르는 도구다

분할을 이해하는 과학적 토대다.

**Schoenfeld·Grgic·Krieger (2019)** 의 빈도 메타분석 결론:
- 볼륨을 동일하게 맞췄을 때(volume-equated), **주 2회가 주 1회보다 근비대에 우수**.
- 부위별 **최소 주 2회** 권장. 주 3회가 주 2회보다 나은지는 결론 미정.

이후 메타분석들이 덧붙인 미묘한 점:
- **볼륨을 똑같이 맞추면 빈도 자체의 추가 효과는 사실상 0에 가깝다**(효과크기 약 0.04). 같은 주간 세트 수라면 주 1회에 몰든 주 3회에 쪼개든 근비대 차이는 미미.
- 빈도를 늘렸을 때 좋아 보이는 경우는 대개 **빈도와 함께 총 볼륨도 늘었기 때문**.
- 빈도의 실용적 가치: **많은 볼륨을 한 세션에 몰지 않고 분산**시켜 세션당 피로를 낮추고 각 세트의 질을 유지.

**실전 번역**
1. 부위별 주 2회를 기본 목표로 삼는다(주 1회보다 안전하게 우수).
2. 주간 볼륨이 커질수록(중·고급) 한 번에 못 하므로 빈도를 늘려 쪼갠다.
3. "빈도 높이면 무조건 더 큰다"는 오해. 빈도는 **볼륨을 효율적으로 배치하는 수단**이다.

이 한 문장이 아래 모든 분할·프로그램 비교의 기준이다: **분할은 곧 빈도 배치 전략이고, 빈도는 볼륨 분배 도구다.**

## 1-3. 분할 종류 비교

| 분할 | 주 훈련일 | 부위별 빈도 | 세션당 부위 볼륨 | 적합 대상 | 주된 목표 |
|---|---|---|---|---|---|
| **풀바디(Full Body)** | 2~3일 | 매 세션(주 2~3회) | 낮음(부위당 3~4세트) | 초보, 시간 부족자 | 근력·기초·습관 |
| **상하체(Upper/Lower)** | 4일 | 주 2회 | 중간 | 중급, 균형 추구 | 근력+근비대 |
| **푸시·풀·레그(PPL)** | 3일 또는 6일 | 3일=주 1회 / 6일=주 2회 | 중~높음 | 중·고급, 시간 여유 | 근비대 중심 |
| **부위별(Bro Split)** | 5~6일 | 주 1회 | 매우 높음 | 고급 보디빌더 | 부위별 고볼륨·집중 |
| **아놀드 분할** | 6일 | 주 2회 | 매우 높음 | 고급 | 초고볼륨 근비대 |

각 분할의 상세 구성·요일별 예시는 [2-B. 근비대 계열](#2-b-근비대-계열) 1부에서 종목까지 풀어 다룬다.

- **풀바디**: 매 세션 전신 → 빈도 자연히 주 2~3회로 높음. 세션당 부위 볼륨은 낮은 게 한계. 초보·시간 부족자의 정답.
- **상하체**: 상/하체로 나눠 부위별 주 2회 + 48~72시간 회복. 풀바디의 빈도와 부위별의 집중을 절충 → 대부분 중급자에게 지속가능성·결과 균형이 가장 좋음.
- **PPL**: 밀기/당기기/다리. **3일이면 주 1회, 6일이면 주 2회** — 이 차이가 결정적. 6일 PPL이 근비대에 유리.
- **부위별(Bro)**: 하루 한두 부위 집중, 주 1회. 신선한 상태에서 고볼륨 가능하나 주 2회 대비 근비대 불리(1-2장 근거).

# 2부. 명명 프로그램 카탈로그

점진적 과부하를 실제로 구현한 대표 프로그램들을 일관된 카드 형식으로 정리한다. 모든 구조·증량 규칙은 출처에서 확인한 값이며 추정은 별도 표기했다.

---

## 2-A. 근력 계열

선형 프로그램의 핵심은 "회복 주기 = 1세션". 매 운동마다 무게를 올리고, 실패가 반복되면 리셋해 다시 쌓는다. 중급은 회복 주기를 1세션에서 "1주"로 늘려 한 주 안에 볼륨·강도를 나눠 배치한다.

### 초급 선형 프로그램

#### StrongLifts 5x5
저자/출처: Mehdi(stronglifts.com), Bill Starr 5x5에서 단순화.

| 항목 | 내용 |
|---|---|
| 분할·빈도 | 전신, 주 3회(비연속일). A/B 번갈아 |
| Workout A | 스쿼트 5x5, 벤치 5x5, 바벨로우 5x5 |
| Workout B | 스쿼트 5x5, 오버헤드프레스 5x5, 데드리프트 1x5 |
| 시작 무게 | 스쿼트·벤치·OHP 빈 바(20kg), 데드 40kg, 로우 30kg |
| 증량 규칙 | 모든 세트 성공 시 다음 세션 스쿼트·벤치·로우·OHP +2.5kg, 데드 +5kg(매 세션) |
| 실패 시 | 같은 무게 반복. **같은 종목 3세션 연속 실패 → 10% 감량(deload) 후 재상승** |

- 대상: 완전 초보 ~ 입문 6개월. 목표: 근력 위주(초기 근비대 동반).
- 강점: 매우 단순, 5대 컴파운드 집중, 빠른 초기 근력·기술 습득.
- 약점: 스쿼트 주 3회로 후반 회복 부담 큼, 상체 보조·팔 부족.

#### Starting Strength
저자/출처: Mark Rippetoe(『Starting Strength』).

| 항목 | 내용 |
|---|---|
| 분할·빈도 | 전신, 주 3회. A/B 번갈아 |
| Workout A | 스쿼트 3x5, 벤치 3x5, 데드 1x5 |
| Workout B | 스쿼트 3x5, OHP 3x5, 데드 1x5(후반 파워클린 5x3 대체 가능) |
| 증량 규칙 | 매 세션 무게 추가. 하체 큰 점프, 상체 작은 점프. 진행되며 점프 폭 축소 |
| 실패 시 | **3세션 연속 실패/정체 시 리셋: 해당 종목 5~10%(보통 10~15%) 감량 후 작은 점프로 재상승** |
| 다중 리셋 | 리셋 2~3회 후 또 정체 → 선형 단계 종료, 중급으로 |

- "Reset"은 프로그래밍 변경이 아니라 단순 5~10% 부하 하향.
- 대상: 완전 초보. 목표: 순수 근력.
- 강점: 적은 종목(3x5)으로 세션당 회복 여유 → SL보다 무게 점프를 더 오래 유지. 코칭 자료 방대.
- 약점: 상체 볼륨·등 운동 부족, 보디빌딩 요소 거의 없음.

#### GreySkull LP (GSLP)
저자/출처: John Sheaffer(Johnny Pain). 인기 변형 "Phrak's GSLP"(로우·친업 추가).

| 항목 | 내용 |
|---|---|
| 분할·빈도 | 전신, 주 3회. A/B 번갈아 |
| 메인 세트 | 컴파운드 **2x5 + 1x5+**(마지막 AMRAP). 데드는 1x5+ 한 세트 |
| 종목 | 스쿼트, 벤치/OHP, 데드/로우 + 끝에 고립(컬·딥 등). Phrak 변형은 매 세션 친업·로우 |
| 증량 규칙 | 스쿼트·데드 +5lb(2.5kg), 벤치·OHP +2.5lb(1.25kg) |
| 가속 증량 | **마지막 AMRAP 10회 이상 → 다음 증량폭 2배** |
| 실패 시 | **AMRAP 5회 미달 → 다음 세션 해당 종목 10% 감량** 후 재상승 |

- 대상: 초보~입문, 근육도 함께 키우려는 사람. 목표: 근력+근비대.
- 강점: AMRAP로 잘하는 날 추가 볼륨, 고립 포함, 두 단계 증량폭으로 OHP 정체 완화.
- 약점: AMRAP 자기조절 필요, 순수 파워리프팅엔 SS보다 덜 특화.

#### Ivysaur 4-4-8
저자/출처: Reddit /u/ivysaur. SL·SS의 "5세트" 방식 개선 의도.

| 항목 | 내용 |
|---|---|
| 분할·빈도 | 전신, 주 3회. Week A / Week B |
| 메인 세트 | 같은 종목을 한 세션은 **4x4**, 다른 세션은 **4x8**(8렙 무게는 4렙의 약 90%) |
| 증량 규칙(주당) | 스쿼트·데드 +15lb, 벤치·로우 +10lb, OHP +5lb |
| 가속 증량 | **그 주 AMRAP 8회 이상 → 다음 주 증량폭 2배** |

- 대상: 초보(SL/SS 대안), 3일 파워리프팅 지향. 목표: 근력+근비대.
- 강점: 무거운 4렙 + 볼륨 8렙 결합, 주 단위 부하 관리로 세션 부담 분산.
- 약점: 주당 큰 점프라 정체 시점이 거칠 수 있음, 구조가 SL보다 복잡.

### 중급 프로그램

#### Madcow 5x5
저자/출처: "Madcow"(EliteFitness 포럼)가 Bill Starr 5x5 개작.

| 항목 | 내용 |
|---|---|
| 분할·빈도 | 전신, 주 3회. HLM(Heavy-Light-Medium) |
| 월(Heavy) | 램핑 5x5 — 1~4세트 점증, 5세트째가 그 주 톱세트 |
| 수(Light) | 가벼운 볼륨(월 톱세트 일부 부하), 회복·기술 |
| 금(Intensity) | 램핑 후 톱세트 1x3 신기록 + 백오프 1x8 |
| 증량 규칙 | **매주 톱세트 약 +2.5%**(주 단위 선형) |
| 온램프 | 1~3주 가볍게, 4주 기존 최고치 복귀, 5주부터 신기록 |

- 대상: 선형 소진한 6~18개월 중급자. 목표: 근력 위주(근비대 동반).
- 강점: 램핑으로 무거운 볼륨 줄여 피로 억제, 매주 신기록 가능. 8~12주 운용.
- 약점: 종목 고정, 보조·약점 보완 여지 적음.

#### Texas Method
저자/출처: Rippetoe & Glenn Pendlay(『Practical Programming』).

| 항목 | 내용 |
|---|---|
| 분할·빈도 | 전신, 주 3회 |
| 월(Volume) | 스쿼트 5x5 @ 금요일 무게의 약 90%. 그 주 가장 힘든 날 |
| 수(Recovery) | 가벼운 저볼륨(스쿼트는 월의 약 80%), OHP 등 |
| 금(Intensity) | 램핑 후 **1x5 신기록(5RM PR)** |
| 증량 규칙 | 금요일 PR 성공 시 다음 주 상체 +5lb, 하체 +10lb |
| 실패 시 | 금요일 미스 → 증량 금지·회복 점검. **2주 연속 미스 → 월 볼륨 1세트 감축**. 더 정체 시 월 부하 5~10% 하향, 또는 금요일 렙 순환(1x5→2x3→3x2→5x1→리셋) |

- 대상: 선형 소진 중급자. 목표: 근력 위주.

## 2-B. 근비대 계열

이름난 근비대 프로그램은 대부분 [1부]의 분할 유형 위에 세부 처방을 얹은 것이다. 먼저 분할 유형의 종목·볼륨 감각을 짚고, 구조화된 저자 프로그램으로 넘어간다.

핵심 용어: **주당 부위 세트 수**(흔한 가이드 부위당 주 10~20세트), **빈도**(주 2회가 주 1회보다 다소 우위), **RIR/RPE**(세트 종료 시 남긴 반복 수 / 인지 강도, RPE 10=실패).

### 기본 분할 유형 — 종목·볼륨 상세

#### PPL (Push / Pull / Legs)
- 6일 버전이면 각 부위 주 2회, 3일이면 주 1회.
- 6일 중급 예시(StrengthLog): Push=벤치·OHP·덤벨플라이·사이드레터럴·삼두익스텐션 / Pull=데드·랫풀다운·바벨로우·바벨컬 / Legs=스쿼트·레그프레스·레그컬·카프. 일정 Push–Pull–Legs ×2.
- 진행: 주 운동 %1RM 기반, 보조는 주차마다 **세트 추가**로 볼륨 점증.
- 주당 부위 볼륨: 약 12세트(기본)~18세트(종목당 +1세트). 대상: 회복 되는 중·상급.

#### Upper / Lower
- 상체 2회 + 하체 2회 = 주 4일, 부위별 주 2회.
- 구성: 복합 3~4세트×6~10회 + 고립 2~3세트×10~15회. Day A 무거운 저반복, Day B 중간 부하 고반복.
- 주당 부위 볼륨: 약 12~16세트. 대상: 초·중급(빈도·회복 균형).

#### Bro Split (1부위 1일)
- 월=가슴, 화=등, 수=어깨, 목=팔, 금=다리. 주 5일, 부위별 주 1회.
- 한 부위 4~6종목, **세션당 15~20+세트** 몰아치기.
- 강점: 부위당 회복 충분, 멘탈 단순, 강한 펌프. 약점: 주 1회는 등량 조건에서 주 2회 대비 성장 다소 열세.

#### Arnold Split
- 6일. ① 가슴+등 ② 어깨+팔 ③ 다리+허리를 주 2회씩. 부위별 주 2회.
- 종목당 워밍업 제외 4~5 작업세트, 피라미드(고→저). 가슴–등 슈퍼세트.
- 주당 부위 볼륨: 20세트 이상. 대상: 회복·영양·경험 갖춘 상급.

#### Full Body 근비대
- 매 세션 전신, 주 3회면 부위별 주 3회. 세션당 부위 1~2종목으로 빈도 분산.
- 주당 부위 볼륨: 약 9~18세트. 대상: 초급·시간 제약자.

  - B: 프론트스쿼트 3×5, OHP 3×5, RDL 3×8, 랫풀다운 3×8, 복근 2×15 + 바벨컬 2×10
- **진행(선형)**: 매주 상체 +5lb, 하체 +10lb. 5회 구간은 증량 안 하는 주 세트당 +1회, 8–15회 구간은 +2회.
- 대상: 초보. 부위 주간 볼륨 약 9~12세트(빈도 높아 효율 좋음).

# 3부. 목표별 훈련 구성

특정 프로그램과 별개로 "이 목표를 원하면 훈련 변수를 어떻게 맞추는가"를 운동과학 합의(ACSM, NSCA)로 정리한 처방이다. 변수 8가지: ① 반복 범위 ② 강도(%1RM) ③ 세트·주간 볼륨 ④ 세트 간 휴식 ⑤ 주당 빈도 ⑥ 템포 ⑦ 점진적 과부하 방식 ⑧ 종목(복합 vs 고립).

> **큰 그림**: 2026 ACSM은 17년 만의 개정에서 "근비대는 특정 '마법의 반복 범위'에 갇혀 있지 않다 — 노력만 충분하면 넓은 부하 스펙트럼에서 성장한다. 다만 **주간 볼륨**이 특히 중요하다"로 무게중심을 옮겼다. 반복 범위는 절대 규칙이 아니라 출발점이라는 게 현재 합의다.

## 3-1. 목표별 핵심 처방표

| 변수 | 최대근력 | 근비대 | 근지구력 | 파워 | 체지방 감량(컷) |
|---|---|---|---|---|---|
| 반복 범위 | 1–6 | 6–12 중심, 5–30 가능 | 15+ | 1–5(폭발적) | 유지(주로 5–12) |
| 강도 %1RM | ≥80–85% | 67–85%(저부하도 노력 충분 시 유효) | <67% | 30–70%(폭발 의도) | 평소 강도 유지(≥70–80% 핵심세트) |
| 세트·주간 볼륨 | 종목당 2–3세트, 근육당 주 ~10세트 이하도 가능 | 근육당 주 10세트+(10–20) | 2–3세트+ | 3–6세트, 적은 반복 | 볼륨 약간 축소·강도 보존 |
| 세트 간 휴식 | 2–5분 | 30초–1.5분(복합 ~2분) | <30초 | 2–5분(완전 회복) | 충분히(2–3분) |
| 주당 빈도 | 근육당 ≥2회 | 근육당 2회+ | 2–3회 | 2–3회 | 2–4회 |
| 템포 | 통제하되 폭발 의도 | 2–8초/반복(느린 이심 강조) | 통제·연속 | 이심 통제+동심 최대속도 | 평소 유지 |
| 종목 비중 | 복합 위주 | 복합+고립 균형 | 복합·고립, 순환 | 폭발적 복합 | 복합 위주(효율) |

## 3-2. 최대근력
- **반복·강도**: NSCA는 근력에 ≥85% 1RM, ACSM 2026은 "≥80% 1RM 완전 가동범위, 세션 초반, 종목당 2–3세트, 주 2회 이상", 반복 통상 ≤6회.
- **휴식**: 신경계·인산크레아틴 회복 위해 2–5분.
- **점진적 과부하**: **부하 증량 1순위**. **2-for-2 룰**(목표보다 2회를 2세션 연속 더 하면 증량) 또는 단순 증분(상체 +1–2.5kg, 하체 +2.5–5kg). 주기화가 비주기화보다 최대근력 우수(메타분석 일관).
- **종목**: 스쿼트·데드·벤치·OHP 복합 위주, 고립은 보조.

## 3-3. 근비대
- **반복·강도**: 전통 6–12회·67–85%이나 ACSM 2026·최신 연구는 **5–30회 전반에서 실패 근처까지 노력하면 성장** — 부하보다 충분한 노력 + 주간 볼륨이 핵심.
- **세트·볼륨**: 근육당 주 ~10세트 기준선, 더 많은 볼륨(10–20)에서 추가 이득 경향(회복 한계 내).
- **휴식**: 30초–1.5분이 전통이나 복합은 다음 세트 볼륨 확보 위해 ~2분이 유리.
- **템포**: 반복당 2–8초, 빠른 동심 + 느린 이심. 극단적 저속은 총 볼륨 깎아 역효과.
- **점진적 과부하**: **이중 진행(double progression)**이 실무 표준 — 반복 범위 상단을 모든 세트에서 채우면 증량 후 하단 복귀. 부하/반복 진행은 볼륨 동일 시 결과 동등.
- **종목**: 복합+고립 균형.

## 3-4. 근지구력
- 15회 이상, <67% 1RM. 2–3세트+, 세트 간 <30초(서킷). 주 2–3회. 통제된 연속 동작.
- **과부하**: 반복 수 증가 → 세트 증가 → 휴식 단축 → 부하 소폭 증가. "지속 시간/총 반복"을 늘리는 방향.

## 3-5. 파워
- **반복·강도**: ACSM 2026은 30–70% 1RM을 동심 구간 최대 속도로, 반복 1–5회(속도 저하 없이).
- **휴식**: 신경계 의존 → 2–5분 완전 회복.
- **템포**: 이심 통제, 동심 최대 의도 속도.
- **과부하**: "동일 부하에서 더 빠르게" 또는 "동일 속도에서 더 무겁게"(속도-부하 곡선 우상향). VBT의 목표 바 속도 유지가 과부하 신호.
- **종목**: 올림픽 변형(클린·스내치·풀), 점프 스쿼트, 메디신볼 던지기.

## 3-6. 체지방 감량(컷) — 근손실 방지 훈련
핵심: **칼로리 적자에서 근육을 지키는 가장 강력한 변수는 "저항운동 + 충분한 단백질"이다.** 저항운동 병행 시 식이만 할 때보다 제지방 손실이 줄고(식이 단독 24% → 병행 11%), 일부 연구는 칼로리 제한 유발 제지방 손실의 93.5%를 저항운동이 상쇄.

- **강도(부하)는 유지**: 적자에서도 평소 무거운 핵심 세트(≥70–80% 1RM) 유지가 근육 보존의 핵심 레버. "가볍게·고반복으로 태운다"는 통념과 반대.
- **볼륨은 회복 가능 범위로 조정**: 적자엔 회복력 저하 → 총 볼륨 약간 축소하되 강도는 유지. 볼륨이 제지방 보존에 미치는 영향은 연구마다 엇갈려 무리한 고볼륨이 필수는 아님.
- **참고**: 일부 메타분석은 칼로리 제한 중 중·저강도가 고강도보다 제지방 보존에 유리하다고도 함(합의 아님). 실무 권고는 "강도는 유지하되 총 부하·실패 빈도는 과욕하지 말 것".
- **휴식**: 강도 보존 위해 2–3분. **빈도**: 주 2–4회, 주요 근육 주 2회+.
- **과부하**: 목표를 "향상"이 아니라 **"유지(성과 보존)"**로 재설정 — 같은 무게×반복을 지켜내면 성공.
- **동반 변수**: 적자 크기(완만할수록 유리), 단백질(체중당 충분히), 수면, 훈련 경력.

## 3-7. 하이브리드 — 근력+근비대 동시
- **개념**: 동시·파동 주기화(concurrent / DUP). 세션마다 강도·반복을 바꿔 강도일과 볼륨일이 같은 회복 자원을 두고 경쟁하지 않게.
- **주간 예(같은 동작)**: Day1 근력 5×3 무겁게(휴식 2–5분) / Day2 근비대 4×8 중간(~1.5–2분) / Day3 파워 6×2 빠르게(2–5분).
- **주간 파동(12주 근비대 블록)**: 1주 10–12회 → 2주 8–10회 → 3주 6–8회, 3주 주기 반복.
- **근거**: 주기화가 비주기화보다 최대근력 우수, 비대는 총량 같으면 모델 간 차이 작음 → "근력은 주기화로, 비대는 볼륨 확보로".

# 4부. 나에게 맞는 훈련 고르기

선택은 4개 축의 교차로 결정된다: **목표 × 경력 × 가용 시간(주 며칠) × 장비.**

## 4-1. 가용 시간(주 훈련일) — 1차 필터

| 주 훈련일 | 1순위 추천 | 대안 | 비고 |
|---|---|---|---|
| **2일** | 풀바디 ×2 | 상/하체 ×2 | 분할 금지, 전신으로 빈도 확보 |
| **3일** | 풀바디 ×3 | PPL(부위별 주 1회) | 초보 기본값 |
| **4일** | 상/하체 ×2 | 풀바디+보조 | 중급 황금 구간, 부위별 주 2회 |
| **5일** | 상/하체 + PPL 하이브리드 | U/L 변형 | 부위별 주 2회 유지 |
| **6일** | PPL ×2 | 아놀드/부위별 | 고볼륨 처리 가능자만 |

## 4-2. 목표 × 경력 통합 결정표

| 목표 ↓ / 경력 → | 초급 | 중급 | 고급 |
|---|---|---|---|
| **근력** | 풀바디 선형진행(주 3일) — SL/SS/GSLP | 상/하체 선형 주기화 — Texas/5·3·1 | 주기화(선형·블록·DUP) — Bullmastiff |
| **근비대** | 풀바디 주 3일 — Fierce 5 | 상/하체 또는 6일 PPL — PHUL/PHAT | 6일 PPL·하이브리드·고볼륨 — RP |
| **감량** | 풀바디 주 3일 + 유산소 | 상/하체 + 유산소 | 익숙한 분할 유지 + 유산소·식이 |

> **감량에는 별도의 "감량용 분할"이 없다.** 분할은 근육 유지·자극용, 감량은 식이(칼로리 적자)가 결정. 평소 분할에 유산소·식이를 더한다.

## 4-3. 장비 기준

| 환경 | 권장 분할 | 이유 |
|---|---|---|
| 풀세팅 헬스장 | 무엇이든 | 바벨·머신·케이블 풍부 |
| 홈(덤벨+밴드) | 풀바디·상/하체 | 종목 제한 → 적은 동작으로 전신 |
| 맨몸 위주 | 풀바디·상/하체 | 부하 종류 적어 빈도로 자극 |

## 4-4. 의사결정 플로우

```
1) 주 며칠 갈 수 있나?
   ├─ 2~3일 → 풀바디로 확정 (빈도 확보 최우선, 분할 금지)
   └─ 4일 이상 → 2단계로

2) 경력은?
   ├─ 초급(약 1년 미만, 선형 증량이 계속 됨)
   │     → 일수와 무관하게 풀바디 선형진행 유지 (4일이어도 OK)
   └─ 중급 이상 → 3단계로

3) 1순위 목표는?
   ├─ 근력          → 상/하체 + 주기화
   ├─ 근비대 + 4일  → 상/하체 (PHUL 등)
   ├─ 근비대 + 5~6일→ PPL 또는 하이브리드 (부위별 주 2회 확인)
   └─ 감량          → 기존 분할 유지 + 유산소/식이

4) 장비로 최종 조정
   └─ 홈/맨몸이면 분할을 풀바디·상하체 쪽으로 한 단계 단순화
```

## 4-5. 초·중·고급 단계별 추천 경로

분할은 "졸업"하며 갈아타는 것이다.

| 단계 | 특징 | 분할 | 진행 방식 |
|---|---|---|---|
| **초급** | 거의 매 세션 무게가 오름 | 풀바디 주 3일 | **선형 진행** — 세션마다 상체 +1~2.5kg, 하체 +2.5~5kg. 동작 학습 핵심. 기본 종목 12주+ 고수 |
| **중급** | 세션마다는 못 오르고 주 단위로 오름 | 상/하체(주 4일) 또는 6일 PPL | **이중 진행** — 반복 범위 상단 채우면 증량. 부위별 주 2회 |
| **고급** | 월 단위로도 진행 느림 | 6일 PPL·하이브리드·부위별 | **주기화** — 선형/DUP/블록으로 볼륨·강도 파동, 정체 회피 |

> 단계를 넘어가는 신호는 "나이"가 아니라 "진행이 멈추는 방식"이다. 세션마다 무게가 더는 안 오르면 → 중급식 이중진행으로, 주 단위로도 막히면 → 주기화로.

## 4-6. 흔한 선택 실수

1. **프로그램 호핑** — 새 루틴을 2~3주 하고 갈아타기를 반복 → "영원한 초보". 점진적 과부하는 같은 동작을 8~16주 추적해야 효과가 드러난다. **한 프로그램 최소 8~12주 고수.**
2. **가용 시간 대비 분할 과다** — 주 3일이면서 6일 PPL을 짜면 빈도가 무너진다. 일수 적을수록 풀바디로.
3. **빈도를 목적으로 착각** — "더 자주 = 더 큰다" 오해. 빈도는 볼륨 분배 수단(1-2장).
4. **모든 변수 동시 증가** — 무게·반복·세트를 한꺼번에 늘려 번아웃. 한 번에 한 변수만.
5. **초보가 부위별(bro) 분할로 시작** — 빈도 주 1회라 학습·근비대 모두 불리. 초보는 풀바디.
6. **단계를 안 넘어가거나 너무 빨리 넘어감** — 선형이 아직 되는데 주기화로 점프, 혹은 막혔는데 같은 방식 고수.

## 부록 2. 용어

- **1RM**: 1회 들 수 있는 최대 중량. **TM(트레이닝 맥스)**: 보통 1RM의 90%, 5/3/1·nSuns 등이 기준으로 사용.
- **AMRAP**: As Many Reps As Possible, 그 세트에서 가능한 최대 반복.
- **RIR / RPE**: 세트 종료 시 남긴 반복 수 / 인지 강도(RPE 10=실패, 9=1회 여유, 8=2회 여유).
- **이중 진행(double progression)**: 정한 반복 범위 상단을 모든 세트에서 채우면 증량 후 하단으로 복귀.
- **램핑(ramping)**: 워밍업처럼 점증해 마지막 톱세트에 도달하는 방식.
- **디로드(deload)**: 부하·볼륨을 일시 낮춰 회복하는 주/구간.
- **볼륨 랜드마크(MV/MEV/MAV/MRV)**: RP의 부위당 주간 세트 기준점(유지/최소효과/최대적응/최대회복가능).
- **DUP(Daily Undulating Periodization)**: 세션마다 강도·반복을 파동시키는 주기화.
- **HLM**: Heavy-Light-Medium 주간 배치(Madcow 등).
