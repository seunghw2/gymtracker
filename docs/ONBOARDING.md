# GymTracker · 온보딩(AI 인테이크) 질문 정리

> AI 애널리스트가 분석 정확도를 위해 받는 인테이크 질문. 채팅형으로 한 질문씩 등장.
> 출처: `app/ai/intake.tsx` · 저장: `PUT /api/v1/ai/profile`(`ai_profile`).
> 표기: **라벨**(사용자에게 보이는 문구) → `저장값`(백엔드 전달 값).

총 **9문항**. 1~6은 단일 선택, 7은 다중 선택, 8은 칩+직접입력, 9는 자유 입력.

---

## 1. 운동 목표 (필수)
**“반가워요! 몇 가지만 빠르게 물어볼게요. 운동 목표가 뭐예요?”**  · 단일 선택 → `goalPhysique`

| 라벨 | 저장값 |
|---|---|
| 💪 근육을 키우고 싶어요 | `lean_muscle` |
| 🏋️ 더 강해지고 싶어요 | `strength` |
| 🔥 체지방을 줄이고 싶어요 | `fat_loss` |
| 📅 운동 습관을 만들고 싶어요 | `habit` |
| ❤️ 건강하게 유지하고 싶어요 | `health` |
| 🤔 잘 모르겠어요 | `unsure` |

## 2. 목표 체중
**“목표 체중이 있나요?”**  · 단일 선택 → `weightGoal`

| 라벨 | 저장값 |
|---|---|
| 증량 | `gain` |
| 유지 | `maintain` |
| 감량 | `loss` |
| 모름 | `unknown` |

## 3. 운동 실력
**“지금 운동 실력은 어느 쪽에 가까워요?”**  · 단일 선택 → `experienceLevel`

| 라벨 | 저장값 |
|---|---|
| 막 시작했어요 | `beginner` |
| 기본기는 있어요 | `intermediate` |
| 베테랑이에요 | `advanced` |

## 4. 운동 경력
**“운동을 꾸준히 한 지는 얼마나 됐어요?”**  · 단일 선택 → `trainingMonths`(개월)

| 라벨 | 저장값 |
|---|---|
| 6개월 미만 | `3` |
| 6개월~1년 | `9` |
| 1~3년 | `24` |
| 3년 이상 | `48` |

## 5. 운동 빈도
**“일주일에 보통 며칠 운동해요?”**  · 단일 선택 → `weeklyFrequencyTarget`(회/주)

| 라벨 | 저장값 |
|---|---|
| 주 2회 | `2` |
| 주 3회 | `3` |
| 주 4회 | `4` |
| 주 5회 | `5` |
| 주 6회+ | `6` |

## 6. 세션 길이
**“한 번 운동하면 보통 얼마나 걸려요?”**  · 단일 선택 → `sessionMinutes`(분)

| 라벨 | 저장값 |
|---|---|
| 30분 | `30` |
| 45분 | `45` |
| 60분 | `60` |
| 90분+ | `90` |

## 7. 우선 부위 (다중 선택)
**“특히 키우고 싶은 부위가 있어요? (여러 개 골라도 돼요)”**  · 다중 선택 → `priorityMuscles[]`

| 라벨 | 저장값 |
|---|---|
| 가슴 | `chest` |
| 등 | `back` |
| 어깨 | `shoulders` |
| 하체 | `legs` |
| 팔 | `arms` |
| 코어 | `core` |

## 8. 통증·제약 (칩 + 직접 입력, 선택)
**“혹시 아프거나 조심해야 할 곳이 있어요? (없으면 넘어가도 돼요)”**  → `constraints[]`

- 칩(복수 선택): 어깨 · 허리 · 무릎 · 손목 · 팔꿈치 · 목
- 직접 입력: 자유 텍스트(예: “왼 어깨가 시큰함”)
- 버튼: **없어요** / **다음**

## 9. 자유 메모 (선택)
**“마지막! 분석할 때 참고하면 좋을 내용이 더 있을까요? (없으면 건너뛰어도 돼요)”**  → `freeNote`

- 자유 텍스트 입력
- 버튼: **건너뛰기** / **저장하고 분석 시작**

---

### 참고
- `splitStyle`(분할 스타일) 질문은 제거됨 — 항상 `null` 저장.
- `weightGoal` 컬럼은 백엔드 재시작 후 `ai_profile.weight_goal`에 저장됨(ddl-auto 자동 생성).
- 기존 프로필이 있으면 각 질문은 이전 답으로 프리필됨.
