# GymTracker · 온보딩(AI 인테이크) 정리

> 원칙: **"AI가 분석하기 위해 정말 필요한 정보만 받는다."** 목표를 넓게가 아니라 **깊게** 묻는다(성공 기준 = 북극성).
> 출처: `app/ai/intake.tsx` · 순수 로직 `lib/onboarding.ts` · 저장 `PUT /api/v1/ai/profile`(`ai_profile`).
> 표기: **라벨**(보이는 문구) → `저장값`. 채팅형, 한 질문씩 등장 → 답변 칩(우측) → 확정 말풍선.

총 **9문항 + 요약 화면**. Q2는 Q1(목표)에 따라 **분기**.

---

## Q1. 운동 목표 (필수)
**“운동하는 가장 큰 이유는 무엇인가요?”** → `goalPhysique`

| 라벨 | 저장값 |
|---|---|
| 💪 근육을 키우고 싶어요 | `lean_muscle` |
| 🏋️ 더 강해지고 싶어요 | `strength` |
| 🔥 체지방을 줄이고 싶어요 | `fat_loss` |
| 📅 운동 습관을 만들고 싶어요 | `habit` |
| ❤️ 건강하게 유지하고 싶어요 | `health` |
| 🤔 잘 모르겠어요 | `unsure` |

## Q2. 성공 기준 (북극성 · 분기) → `successGoal`
Q1에 따라 질문과 선택지가 달라진다.

- **근육(lean_muscle)** — “6개월 뒤 가장 이루고 싶은 것은?”
  어깨 넓어지기 `shoulders_width` / 등 넓어지기 `back_width` / 상체 키우기 `upper_body` / 하체 키우기 `lower_body` / 전신 균형 `balance`
- **근력(strength)** — “가장 강해지고 싶은 운동은?”
  벤치프레스 `bench` / 스쿼트 `squat` / 데드리프트 `deadlift` / 턱걸이 `pullup` / 전반적 중량 `overall_load`
- **감량(fat_loss)** — “감량에서 가장 중요한 것은?”
  체중 감량 `weight` / 체지방 감소 `bodyfat` / 복부 지방 `belly` / 근손실 최소화 `preserve_muscle`
- **그 외(habit·health·unsure) 공통** — “6개월 뒤 가장 이루고 싶은 것은?”
  꾸준함 유지 `consistency` / 컨디션 개선 `condition` / 체형 관리 `shape` / 전반적 건강 `health` / 아직 모름 `unsure`

## Q3. 우선 부위 (다중) → `priorityMuscles[]`
**“특히 집중하고 싶은 부위가 있나요?”** — 가슴 `chest` · 등 `back` · 어깨 `shoulders` · 하체 `legs` · 팔 `arms` · 코어 `core`

## Q4. 운동 빈도 → `weeklyFrequencyTarget`
**“일주일에 몇 번 운동 가능하신가요?”** — 2 / 3 / 4 / 5 / 6(=6회 이상)

## Q5. 세션 길이 → `sessionMinutes`
**“한 번 운동 시 얼마나 가능하신가요?”** — 30 / 45 / 60 / 90

## Q6. 현재 체중 방향 → `weightGoal`
**“현재 체중은 어느 쪽인가요?”** — 증량 중 `gain` / 유지 중 `maintain` / 감량 중 `loss` / 잘 모르겠음 `unknown`

## Q7. 운동 경력 → `trainingMonths`(개월)
**“운동 경력이 얼마나 되었나요?”** — 6개월 미만 `3` / 6개월~1년 `9` / 1~3년 `24` / 3년 이상 `48`
- `experienceLevel`은 코드에서 도출: `<6 beginner · <36 intermediate · ≥36 advanced`.

## Q8. 통증·제약 (칩 + 입력) → `constraints[]`
**“현재 운동에 영향을 주는 통증이 있나요?”** — 칩: 어깨·허리·무릎·손목·팔꿈치·목 + 하단 입력바로 직접 입력 + **없어요/다음**.

## Q9. 자유 메모 (선택) → `freeNote`
**“AI에게 알려주고 싶은 점이 있을까요?”** — 하단 입력바로 입력 + **건너뛰기/다음**.

## 마지막 · ‘AI가 이해한 목표’ 요약 화면
온보딩 끝에 정리 카드를 보여주고 확인받는다.

```
🎯 AI가 이해한 목표
주 목표 : 근육 키우기
세부 목표 : 어깨 넓어지기      ← 북극성(빨강 강조)
집중 부위 : 어깨, 등
운동 가능 : 주 4회 / 60분
체중 : 유지 중
경력 : 1~3년
주의 : 허리
앞으로 모든 브리핑과 리포트는 이 목표를 기준으로 분석할게요.
[수정할게요]   [좋아요]
```
- **좋아요** → `putAiProfile` 저장 후 `/ai` 진입.
- **수정할게요** → 답변 유지한 채 처음부터 다시(칩 프리필).

### 참고
- `split`(분할) 질문은 제거됨 — 기록에서 파악 가능.
- `weightGoal`·`successGoal` 컬럼은 백엔드 재시작 후 `ai_profile`에 생성(ddl-auto). 둘 다 리포트 LLM 입력(Profile)에 포함돼 AI가 세부 목표 기준으로 해석.
- UI: 답변 칩 = 우측 균일폭 세로 스택·중립 다크(빨강은 보낸 말풍선에만), 하단 입력바 상시.
